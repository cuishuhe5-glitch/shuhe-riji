常见疑问
截图会被长期保存吗？

隐私模式开启时，截图只用于当次识别，识别完成后不会进入长期留存。

数据会上传到云端吗？

活动记录、报告、聊天历史保存在本机 SQLite 数据库；AI 请求直连你配置的模型网关。

API Key 存在哪里？

macOS 上优先保存到钥匙串；面板不会回显完整密钥。

工作记录、报告和聊天历史只保存在本机数据库。
